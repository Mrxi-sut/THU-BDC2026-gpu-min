# 1、改动点
以前是所有因子每天一起投票；
现在是先看今天什么行情，再让擅长这种行情的因子投票。
这样可以减少弱因子拖后腿，同时保留某些因子在特殊行情里的优势。
方案一不是“再加更多因子”，而是让因子有条件上场：
industry_core 这类因子不是永久好或永久坏，而是只在它历史上擅长的市场状态里参与排序；不擅长的时候就关闭，避免污染 selector_score。
# 2、改动点
一句话概括：方案二不是换一个更复杂的模型，而是把调参目标从“预测收益”改成“比赛组合收益”，先在现有 tabular selector 主线上做低风险优化。
# 3、改进点
以前是“模型说哪两只股票最好，我就固定买它们，比如 60% 买第一名、40% 买第二名”。
现在变成“模型先告诉我最看好的股票名单，但最后买多少，还要看这些股票本身有多危险、彼此是不是一起涨跌”。
